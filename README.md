# Real - Time Chat App

A full - stack real - time chat application built with modern web technologies. Users can join chat rooms, send messages instalnly, see who is online and get real - time typing and messages status updates. The application is also fully responsive and works on any device.

## Live Demo



## Features 

* Join room chat with username

* Real - Time messaging (Socket.io)

* Shown users that joined the specific room

* online users indicator

* Responsive design

* message status (delivered/seen)

* typing indicator

## Tech Stack

### Frontend 

* React

* TypeScript

* Tailwind CSS

* Zustand

* Socket.io Client

### Backend 

* Node.js

* Express

* Socket.io

## Installation & Setup

Follow these steps to run the project locally

### 1. Clone the repository

git clone https://github.com/Marioboseski/chat-app.git

cd chat-app

### 2. Install dependencies 

#### Frontend 

cd frontend

npm install

#### Backend

cd backend

npm install

### 3. Configure environment

The frontend must run on: 

http://localhost:5173

and the backend on:

http://localhost:5000

### 4. How to run the app

#### Start backend

cd backend

npm run dev

#### Start frontend

cd frontend 

npm run dev
